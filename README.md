## 🛠️ How to Use
1. Clone this repository:  
   `git clone https://github.com/yourusername/cs-resources.git`
2. Explore markdown files in each category
3. Suggest new resources via Issues or Pull Requests

## 🤝 Contribution Guide
Your contributions make this repository better! Here's how to contribute:
1. Fork the repository
2. Create new branch: `git checkout -b new-resource`
3. Add your resource with proper documentation
4. Submit a Pull Request
